## This is a simple player. Drag and drop files to play them.

## Build:
  - ```$ make``` to build the project.

## Run:
  - ```$ ./play``` to run the project after building.

## Supported formats:
  - .wav
  - .ogg
  - .mp3
  - .qoa
  - .xm
  - .mod
